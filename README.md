Template Appwrite Function untuk integrasi **WhatsApp Business Cloud API webhook**.

---

## 🚀 Features

- ✅ Verify webhook Meta WhatsApp Cloud API
- ✅ Receive incoming messages (GET + POST)
- ✅ Auto reply message
- ✅ Ready for Appwrite Functions (Node.js 22)
- ✅ Simple, production-ready structure
